# QA Automation Interview Master Series

Welcome to the documentation site for the QA Automation Interview Master Series.

This project is an original technical book series for QA automation interview preparation. It is designed to help candidates build clear explanations, practical project understanding, and interview-ready confidence across automation testing topics.

## Available Study Tracks

- Selenium WebDriver with Java
- Playwright with TypeScript
- API testing
- SQL for QA engineers
- Git, Jenkins, and CI workflows

## Current Focus

The first volume is **Selenium WebDriver + Java (2026)**. It covers Java fundamentals, Selenium WebDriver, TestNG, framework design, REST Assured, SQL, Git, Jenkins, scenario-based questions, and HR preparation.

## Project Standards

The series follows a consistent writing model:

- Clean Markdown
- Original explanations
- Interview-ready answers
- Practical examples
- Professional technical style
- Clear revision notes

Use the authoring templates and style guide to keep future chapters consistent.
