# QA Automation Interview Master Series

A professional documentation series for QA automation interview preparation, built as an original, structured handbook collection.

## Volume 1: Selenium WebDriver + Java (2026)

Volume 1 focuses on Selenium WebDriver with Java and the surrounding skills expected from QA automation engineers in modern interview loops. The content is organized from language fundamentals through framework design, API testing, database validation, CI workflows, scenario practice, and HR preparation.
